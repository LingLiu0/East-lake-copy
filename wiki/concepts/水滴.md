# 水滴

## 定义

"水滴"是智能体三层架构的最底层，源自《三体》中水滴探测器——"绝对高效的物理工具调用"。

在智能体架构中，水滴是**纯粹的执行机器**，负责执行单一任务：
- 调用工具（API、MCP、函数）
- 模型生成（文本、代码、图像）
- 任何原子化的操作

## 特征

- **极致高效**：没有感情、没有犹豫、没有冗余信息
- **原子化**：只做一件事，做好一件事
- **可替换**：可以快速替换为更好的实现

## 三层架构总结

```
┌─────────────────────────────────────┐
│           智子（管家）               │  ← 理解需求、价值观对齐、记忆管理
├─────────────────────────────────────┤
│           维德（流程）               │  ← 任务分解、流程调度、接口串联
├─────────────────────────────────────┤
│           水滴（执行）               │  ← 工具调用、模型生成、原子操作
└─────────────────────────────────────┘
```
