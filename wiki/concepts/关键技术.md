# 关键技术

## 定义

在AI推理工程化领域，关键技术涵盖从算法到硬件的全栈优化技术。

## 核心技术分类

### 算法层

| 技术 | 说明 |
|------|------|
| **MLA** | Multi-head Latent Attention，多头潜在注意力，通过低秩投影压缩KV Cache |
| **量化** | FP8/INT8/FP4 精度压缩，降低显存和计算成本 |
| **推测解码** | Speculative Decoding，用小模型猜、大模型验加速推理 |

### 系统层

| 技术 | 说明 |
|------|------|
| **PagedAttention** | 借鉴虚拟内存分页机制管理KV Cache |
| **RadixAttention** | 基数树管理KV Cache，复用公共前缀 |
| **PD分离** | Prefill-Decode 分离架构，不同阶段用不同硬件 |

### 框架层

| 技术 | 说明 |
|------|------|
| **vLLM** | PagedAttention开源实现，通用场景首选 |
| **SGLang** | RadixAttention实现，高并发Agent场景首选 |
| **TensorRT-LLM** | NVIDIA深度优化，极致性能 |
