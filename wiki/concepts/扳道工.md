# 扳道工

## 定义

"扳道工"是 Agentic Engineering（智能体工程）中的核心隐喻，源自马克思·韦伯的社会学理论。

在韦伯的原意中：**"理念"创造的"世界观"常常以扳道工的身份规定着轨道，在这些轨道上，利益的动力驱动着行动。**

应用到 AI Agent 领域：
- **扳道工**：设定轨道和规则，规定 Agent 的行为边界
- **轨道**：Agent 必须遵循的工作流程和规范
- **利益**：驱动 Agent 执行的奖励机制和目标

## 核心原则

1. **理念先行**：先用规范文档（如 SPEC、PRD）把需求说清楚
2. **正反馈机制**：建立驱动模型向前的激励机制
3. **停机条件**：设定边界，防止 Agent 失控
