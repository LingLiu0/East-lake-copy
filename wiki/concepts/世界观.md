# 世界观

## 定义

世界观（Worldview）在 Agentic Engineering 中指**约束 Agent 行为的价值观和原则体系**。

源自马克思·韦伯的理论：理念创造的"世界观"以扳道工的身份规定着轨道。

## Agent 世界观的组成

1. **核心价值观**：如 "helpful, harmless, honest"
2. **行为准则**：什么能做，什么不能做
3. **优先级**：当多个目标冲突时的决策顺序
4. **边界约束**：安全底线、红线

## 世界观的实现示例

```yaml
# Agent 行为控制架构
agent_system:
  # 1. 利益引擎 - 驱动行动
  dynamic_interests:
    material_interests: [token优化, 延迟最小化]
    ideal_interests: [用户满意度, 任务准确率]
  
  # 2. 扳道工 - 规定轨道
  switchman_constraints:
    core_worldview: "ethical_helpful_assistant"
    tracks:
      - safety: 任何危害人类的行动 → 拒绝
      - truthfulness: 置信度 < 0.65 → 诚实表达不确定
      - neutrality: 避免偏见和歧视
  
  # 3. 行动控制器 - 轨道上的执行
  action_execution_loop:
    - 生成候选行动
    - 应用扳道工路由
    - 执行或重路由
```
