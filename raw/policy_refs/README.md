# 政策简报参考库

> 本文件夹存放政策简报示例，供AI问答生成时学习参考

## 格式说明

政策简报格式如下：

```markdown
# 政策简报 YYYY-MM-DD

> 重点关注：中央重大会议规划、新质生产力领域、湖北省重点政策

---

## 一、重大政策

### 1. 政策标题
- 来源: 来源网站
- 日期: YYYY-MM-DD
- 摘要: 政策要点摘要...
- [原文](URL)

## 二、新质生产力

### 1. 政策标题
...

## 三、湖北动态

### 1. 政策标题
...
```

## 筛选原则

1. **湖北移动视角**：关注与通信、数字化、产业发展相关的政策
2. **优先保留**：
   - 湖北政策（特别是数字经济、5G、AI相关）
   - 中央重大政策（新质生产力、数字经济等）
3. **排除**：
   - 其他省份地方新闻
   - 个人/荣誉类新闻
   - 民生类、流量类新闻
   - 非发展相关的会议/活动

## 文件命名

`YYYY-MM-DD-政策简报.md`
