# 📥 成果投递箱

> **团队成果投递入口** - 把文件拖到这里，其他交给系统

---

## 🎯 使用方法

### 只需一步：把文件拖进来！

1. **复制**你的成果文件（PDF/Word/图片等）
2. **粘贴**到这个文件夹
3. **完成**！系统会自动处理

---

## 📋 支持的文件类型

| 类型 | 扩展名 | 说明 |
|------|--------|------|
| PDF 文档 | `.pdf` | 自动提取文字和元数据 |
| Word 文档 | `.docx` | 自动提取内容 |
| Markdown | `.md` | 直接导入 |
| 文本文件 | `.txt` | 纯文本 |
| 图片 | `.png`, `.jpg` | 保留原文件 |

## 📁 文件命名建议

为获得最佳效果，文件名建议包含：

```
# 推荐格式
2025-05-周报-张三.pdf
RAG-技术研究报告-李四.docx
产品设计稿-v2.png
```

系统会自动识别：
- 📅 日期信息 → 笔记日期
- 👤 作者信息 → 作者字段  
- 🏷️ 分类关键词 → 自动归类

## ❓ 常见问题

### 文件放进去后去哪里了？
文件会被移动到 `_archive` 文件夹备份，生成的笔记在 `achievements/` 目录。

### 如何查看我提交的内容？
在 `achievements/` 目录下按日期或分类查找。

### 处理失败怎么办？
检查文件是否在 `_inbox` 文件夹中，如果是请联系管理员。
