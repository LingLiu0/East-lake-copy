# AI Agent 是什么？

AI Agent（人工智能代理）是一种能够**自主决策、规划和执行任务**的智能系统。

## 与传统聊机器人的区别

与传统“一问一答”模式不同，AI Agent 具有以下核心特征：

| 特征 | 说明 |
|------|------|
| **自主性** | 能够根据目标自主规划行动步骤 |
| **多步推理** | 通过 Chain-of-Thought 实现复杂推理链 |
| **工具调用** | 能够调用外部工具（搜索、API、执行代码等） |
| **状态管理** | 能够在多轮交互中保持上下文状态 |
| **反思能力** | 能够评估自身输出的质量并进行修正 |

## 背景

AI Agent 的崛起是推理工程化发展的核心驱动力，被视为 **2026年AI三大趋势之一**（另外两个趋势是：从训练到推理、全栈协同设计）。
