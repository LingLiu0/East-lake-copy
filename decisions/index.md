
# 决策记录索引

本页索引所有重要技术和管理决策（ADR）。建议每条决策均基于统一模板（见 templates/decision-template.md）创建，便于团队追溯和AI分析。

## 决策列表
| 编号 | 标题 | 状态 | 日期 |
|------|------|------|------|
| - | - | - | - |

## 状态说明
- `proposed` - 提议中，待讨论
- `accepted` - 已接受，正在执行
- `deprecated` - 已废弃，不再适用

## 创建新决策
1. 复制 `templates/decision-template.md`
2. 重命名为 `NNN-short-title.md`（NNN 为编号）
3. 填写内容
4. 提交 PR 审核

---

## 自动化建议
- 可开发脚本自动扫描 decisions 目录，自动汇总决策列表。
- 支持 embedding 生成与AI检索。

## 决策原则
1. 记录背景和原因，不只是结论
2. 列出考虑过的所有方案
3. 说明选择理由
4. 记录影响范围和后续行动
